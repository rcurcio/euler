# Project Euler Solution

The solutions are currently represented by three
distinct languages:
* Python
* Javascript
* C++

To run the Python solutions, execute
```bash
python filename.py
```

To run the Javascript solutions, install node.js
and execute:
```bash
node filename.js
```

To run the C++ solutions, install g++ and execute:
```bash
g++ filename.c
./filename.out
```
