# Project Euler Solution

The solutions are currently represented by three
distinct languages:
* Python
* Javascript
* C++

To run the Python solution, execute
```bash
python python.py
```

To run the Javascript solution, install node.js
and execute:
```bash
node javascript.js
```

To run the C++ solution, install g++ and execute:
```bash
g++ c++.c
./a.out
```
